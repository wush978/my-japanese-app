# my-japanese-app

## Prompt

```
請根據我指定目標，嚴格遵守以下格式要求，將結果輸出成一個 JSON 陣列。

【格式要求】
1. 必須是合法的 JSON 陣列（Array of Objects），請不要輸出任何解釋性的文字或對話，直接輸出 JSON 即可。
2. 每個單字物件必須包含兩個固定鍵值："jp" 與 "ch"。
3. "jp" 欄位請填入日文單字。如果該單字包含漢字，請務必在單字後方加上半形括號 `()`，並在括號內標註平假名或羅馬拼音（例如："集まって (あつまって)" 或是 "パソコン (Pasokon)"）。
4. "ch" 欄位請填入繁體中文翻譯。翻譯必須盡量精簡、核心，因為這些文字會顯示在字卡上，不宜過長。

【JSON 輸出範例】
[
  {"jp": "りんご", "ch": "蘋果"},
  {"jp": "集まって (あつまって)", "ch": "集合"},
  {"jp": "デスク (Desuku)", "ch": "書桌、辦公桌"}
]
```
